# 📊 HR Data Analysis with Python

A Python-based Exploratory Data Analysis (EDA) project that analyzes Human Resources (HR) data to uncover workforce trends, employee demographics, salary distribution, departmental insights, and other key business metrics using data visualization techniques.

---

## 📌 Project Overview

This project demonstrates the complete process of analyzing HR data using Python. It includes data cleaning, preprocessing, visualization, and insight generation to help organizations make data-driven HR decisions.

---

## ✨ Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Employee Demographics Analysis
- Department-wise Employee Distribution
- Salary Distribution Analysis
- Work Mode Analysis
- Experience Analysis
- Interactive Data Visualizations
- Business Insights

---

## 🛠️ Tech Stack

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

```bash

```
